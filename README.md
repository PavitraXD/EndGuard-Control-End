## 🛡️ EndGuard

Control the End like a pro ⚡

### 🔥 What it does

* 🔒 Lock / 🔓 Unlock End instantly
* 🚫 Block portals & teleports when locked
* 🏃 Force players out of End
* 🐉 Toggle Ender Dragon (ON/OFF)
* 🧊 Peace mode + freeze dragon
* 📊 Bossbar shows dragon status
* ⏱️ Set timer for auto actions
* 💾 Saves state after restart
* 🌍 Multiworld support

### 🎮 Commands

```
/end lock
/end unlock
/end dragon on
/end dragon off
/end status
/end timer <minutes>
```

### 🔑 Permission

```
end.admin
```

### ⚙️ Platform

* Paper 1.21.x

### 🧠 Key Features

* 🔄 Separate End lock & dragon control
* 🚫 Portal/teleport blocking
* 🛟 Safe player evacuation
* 🛡️ Dragon damage + heal control
* 📍 Custom teleport location
* 🔊 Messages, sounds & titles

### 📦 Install

1. Drop jar in `plugins`
2. Start server
3. Edit config (optional)

### ⚠️ Notes

* End 🔓 ≠ Dragon active
* End 🔒 = no entry + players removed
